256 Color Support in Gnome-terminal

May 4th, 2013

To get the fancy color stuff that powerline provides and the great color schemes available for vim on gnome-terminal, you will have to ensure that $TERM is reported correctly. Under Profile Preferences > Title and Command > Custom Command set env TERM=xterm-256color /bin/bash.
